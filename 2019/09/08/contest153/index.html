LeetCode 1186 Maximum Subarray Sum with One Deletion

原题链接

题目大意，求一个序列中，任意一个子序列的最大值，可以从中去掉一个最小值。

这个题目是一个比较好的题目，技巧性也很强。在一个序列中，他的子序列有很多，如果要把所有子序列都找出来，然后再去掉最小值，这种做法就非常
的慢，而且中间有很多重叠的部分。换个思路来思考这个题目，这个最大值是在序列中某一个点，左右两边的连续子序列的最大和构成的。

LeetCode 1187 Make Array Strictly Increasing

原题链接

题目大意，将arr1变成一个严格递增的序列，其中不符合要求的部分可以用arr2中的元素替代，使其递增。

这个题目有些难度，其中有一些过程必须要想的很清楚才能做得出来，首先是替换操作的部分，其实这部分的操作我个人感觉是属于贪心的思想，因为
每一次替换操作，我们希望都是从arr2中选一个尽可能小的一个元素，并且还要满足arr1的严格递增的要求。这一部分还需要一些搜索过程，其实这个
搜索过程我们可以用二分查找来优化。另一部分就是这道题目主要考察的动态规划部分，为什么是动态规划呢，因为在arr1中，我们不确定到某一个位置之前的部分，到底是怎么操作的，到底是在什么位置进行替换的，所以会有
很多种可能性，我们需要从子问题中找出一种最合适的状态，来找出最终的最优解。具体的DP的细节，就是每一个位置，我们可以将他替换，或者保留
当前的元素，如果都符合要求，那么我需要不断的更新dp中的状态，直到整个数组扫描结束即可。这算是一种状态dp吧，和我们经常见到的1维dp或者
2维数组dp不太一样。看代码。
